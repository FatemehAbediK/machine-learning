# Machine_learning_Basics


### 1. Regression_models:
This project demonstrates regression modeling on a housing dataset to predict property prices based on features such as area, number of bedrooms, bathrooms, and amenities. It includes linear regression, gradient descent (SGD), and polynomial regression, along with feature scaling and preprocessing of categorical variables. Model performance is evaluated using RMSE, and results are compared across different approaches to understand model accuracy.


### 2. classification_models_cancer_dataset:
This project applies multiple machine learning models (Logistic Regression, KNN, SVM, Naive Bayes, Decision Tree, and Random Forest) to the Breast Cancer dataset to classify tumors as malignant or benign, with preprocessing, scaling, and hyperparameter tuning used to compare model performance,using accuracy, confusion matrix, and classification report.


### 3. data_prep_regularization:
a comparison of Ridge/lasso/Elastic net with and without gridsearchCV for housing prices in linear model


### 4. fs:
feature selection with SVM model for breat_cancer dataset of sklearn
